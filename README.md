# MainProject


Steps to Running this project locally.

1. git clone https://github.com/Melcolm/angularjs-springboot-mvc.git
2. cd into cloned directory
3. npm install
4. npm start
5. open the browser and go to http://localhost:4200/ (start using the APP)

Note* you will need to install angular CLI, Node JS and git in order clone the Repo.
